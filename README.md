Šálek inspirace — statická šablona webu

Tento balíček obsahuje jednoduchou statickou šablonu pro prezentaci série webinářů "Šálek inspirace".

Struktura:
- index.html  — hlavní stránka
- css/styles.css
- js/main.js
- img/ — složka s obrázky (nahraď vlastními)

Rychlé spuštění:
1) Rozbalte složku.
2) Otevřete index.html lokálně nebo nahrajte do repozitáře GitHub a zapněte GitHub Pages.
3) V souboru index.html upravte meta tagy (og:url, og:image) a action formuláře (Formspree ID).

Formulář:
- Ve výchozím nastavení používá Formspree. Nahraďte akci vlastním endpointem nebo použijte Google Forms.
